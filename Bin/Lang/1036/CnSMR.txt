TCnAboutForm.Label2.Caption=Bienvenue dans l'Analyseur de Relations des Assistants EDI CnPack.<BR><BR>Cet outil est utilisé pour analyser les fichiers Exécutables générés par Delphi et en obtenir les informations sur les unités et les paquets. Il peut également obtenir les relations entre les paquets, et les relations entre les paquets et leurs unités.<BR><BR>Le résultat de l'analyse peut être sauvegardé dans un fichier ARF. Et nous pouvons créer un fichier SMR à partir d'un fichier ARF pour Obtenir/Voir les Relations entre les Exécutables et les Fichiers Sources par la suite.<BR><BR>Par exemple, vous pouvez ouvrir un ou plusieurs Fichiers Exécutables dans la page "Analyser Fichiers Exécutables" et les analyser. Le résultat de l'analyse peut ensuite être sauvegardé dans un fichier ARF et réouvert dans la page "Voir Résultat de l'Analyse".<BR><BR>Le fichier ARF est généré à partir du Résultat de l'Analyse, which only Contains the Relations between the Executables and Pascal Source File NAME, NO Path Information and Other File Types. For Advanced Situation, You can Open an ARF File in "Edit Source-Module Relations" Page to Edit it and Generate an SMR File, which Contains the Relations between the Executables and Full Path Information of Source Files. Then You can View SMR Files in "View Source-Module Relations" Page Later.
